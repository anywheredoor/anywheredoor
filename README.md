My background combines biomedical research with programming and quantitative analysis. I enjoy applying these skills to practical problems, particularly at the intersection of healthcare and technology.

## Featured Projects

- **[Deep Learning for Reduced-LVEF Detection](https://github.com/anywheredoor/phonocardiogram-lvef-deeplearning)** — My final-year project investigated reduced-LVEF detection from smartphone and digital-stethoscope heart sounds using deep learning. The project received an overall grade of **A**. [Read or download my dissertation](https://github.com/anywheredoor/phonocardiogram-lvef-deeplearning/blob/main/dissertation/Final%20Dissertation%20-%20Wonjun%20Yang.pdf).

- **[MyPet World](https://github.com/anywheredoor/MyPet-World)** — A terminal-based virtual-pet game developed in C++ with ncurses as a group project. It includes minigames, inventory management, persistent saves, and difficulty-based gameplay. The project received a perfect course score.

- **[Is Bitcoin Closer to Wall Street or Gold Bars?](https://github.com/anywheredoor/Is-Bitcoin-Closer-to-Wall-Street-or-Gold-Bars)** — An R-based study comparing Bitcoin with the Nasdaq-100 and gold using daily returns and rolling correlations from 2020–2024.
